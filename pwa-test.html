PWA Diagnostic

Checks why "Add to Home Screen" isn't appearing.

How to test on Android

  1. Open this page in Chrome (not Instagram/Snapchat browser)
  2. Open Chrome menu (⋯) → "Add to Home screen" should appear
  3. If not, open DevTools → Application → Manifest
  4. Look for red errors under "Identity" and "Presentation"

Quick Fixes

  • Icons 404? Run python generate_icons.py and upload the icons/ folder.
  • Manifest 404? Ensure /manifest.json is at root, not /public/manifest.json.
  • Not HTTPS? PWA install requires HTTPS (Vercel/Netlify provide this).
  • Still nothing? Visit the site 2–3 times or wait 30s. Chrome has an engagement heuristic.
